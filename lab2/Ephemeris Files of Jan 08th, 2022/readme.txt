Almanac-SatPos-PRN12.txt: Satellite ephemeris of PRN12 derived from Almanac with intervals of 15 min for January 08, 2022
Format: Seconds of GPS Week 2191, ECEF X (m), ECEF Y (m), ECEF Z (m)

----------------------------------------------------------------------------
brdc0080.22n: GPS broadcast navigation ephemeris in RINEX format for January 08, 2022
Format: RINEX 2 (ftp://igs.org/pub/data/format/rinex211.txt). You can read ephemeris parameters using the provided C++ code.

----------------------------------------------------------------------------
igs21916.sp3: Observed precise ephemeris, to be used as the references for computing the orbit errors for January 08, 2022
Format: sp3 (ftp://igs.org/pub/data/format/sp3c.txt). You can read satellite positions using the provided MATLAB 'readsp3' function.

----------------------------------------------------------------------------
Each IGU file contain ephemeris data for two consecutive days. The first half for 'observed' ephemeris, and the second half for 'predicted' ephemeris.

igu21916_00.sp3: Predicted ultra-rapid ephemeris, for January 08, 2022 ("2191 indicates GPS week 2191, and GPS day 6 means it is Friday")
igu21920_00.sp3: Observed ultra-rapid ephemeris, for January 08, 2022 ("2192 indicates GPS week 2192, and GPS day 0 means it is Saturday")
Format: sp3 (ftp://igs.org/pub/data/format/sp3c.txt). You can read satellite positions using the provided MATLAB 'readsp3' function.
